# AlexandrenkoTicket10
 AlexandrenkoTicket10 - Приложение сделанное по критериям экзаменнационного билета номер 10, которое осуществляет вычисления, находя максимальную прибыль, цену при макс прибыли, спрос, спрос для оптимальной цены, исходя из себестоимости товаров и респондентов

 Работу выполнил александренко Михаил 31ИС
